The Portfolio of Thomas Bouffard

ChessCubed
Chesscubed is an online-multiplayer game of chess extrapolated into the 3rd dimension. It can be played with a VR headset (HTC Vive and Oculus Quest tested) in a WebXR compatible browser. It can also be spectated directly in any modern browser. The game is written in Javascript and is built on the A-Frame framework and the underlying THREE.js library, combined with my prior experience using Socket.io for real-time multiplayer communication.
Particle Simulation
Particle Sim is a multiplayer sandbox particle simulator. Originally a class project to make something interesting in canvas, I choose to re-create the sand simulators that were popular back in the early to mid 2000s (of which most are gone or deprecated) in HTML5. I later took a class on real time interaction and have been using what I learned about Node.js and Socket.io to rebuild the first project as a real-time multiplayer experience.

Falling Sand
This is the result of the original project mentioned in Particle Simulation. This locally run proof of concept effectively demonstrates the system unburdened by network constraints. The system works by extracting the ImageData from the current frame and iterating through the pixels. Each element is identified by its color and its behavior is used to determine what the next frame will look like. When the frame is built, the image is pushed back to the canvas ImageData object.

Rocket Jump
Rocket Jump was a team project inspired by the rocket mechanics in many first person combat games, the rocket launcher creates explosive projectiles that exert a force on the player. A popular exploit is to jump and fire a rocket at the ground, therefore launching yourself into the air. We decided to use this as a mechanic along with an implosion counterpart as the main mechanics for an obstacle course style game.

Toss Up
Toss Up is a team project developed as a mobile game. The objective of the game is to defend your castle by flicking away the invading army of stick figures as they charge forward. It was developed to run on an iPad mini and NVIDIA Shield.
A-Star demonstration
While not a game itself, this scene is a demo of the AStar (A*) algorithm at work. characters spawn in the large room and use the algorithm to plan the path they would take to their randomly assigned rooms. I created this implementation in C# and used Unity to visualize the results.
Phase Jump
Phase Jump is a game I designed for a level design class. The core mechanic is the ability to phase through solid objects and the level is intended to be an introduction level to get the player used to solving problems by means of the mechanic while challenging preconceived notions about obstacles. This gray box prototype was built using Construct 2.
