Particle Simulation
This is a multiplayer sandbox particle simulator. Originally a class project to make something interesting in canvas, I choose to re-create the sand simulators that were popular back in the early to mid 2000s (of which most are gone or deprecated) in HTML5. I later took a class on real time interaction and have been using what I learned about Node.js and Socket.io to rebuild the first project as a real-time multiplayer experience.

Falling Sand
This is the results of the original project mentioned in Particle Simulation. It is single player only, but shows how fast the system can work if network speed isn't an issue. The system works by extracting the ImageData from the current frame and iterating through the pixels. Each sand type is identified by its color and its behavior is used to determine what the next frame will look like. When the frame is built, the image is pushed back to the canvas ImageData object.
